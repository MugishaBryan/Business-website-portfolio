Flatland: Evolution of an Interactive Square

Version 1: Basic Background Change

<script> document.body.style.background = "green"; </script>

This version simply changes the entire page background to green on load.

Version 2: Click to Change Color

<div id="square" onclick="document.getElementById('square').style.background = 'green'"></div>

Now the square changes to green when clicked.

Version 3: Mouse Events with Separate Functions

<div id="square" onclick="makeGreen()" onmouseover="makeGray()" onmouseout="makeRed()"></div> <script> function makeGreen() { document.getElementById('square').style.background = "green"; } function makeGray() { document.getElementById('square').style.background = "gray"; } function makeRed() { document.getElementById('square').style.background = "red"; } </script>

Added mouseover (gray) and mouseout (red) events with separate functions.

Version 4: Consolidated Color Function

<div id="square" onclick="changeColour('green')" onmouseover="changeColour('gray')" onmouseout="changeColour('red')"></div> <script> let square = document.getElementById('square'); function changeColour(colour) { square.style.background = colour; } </script>

Consolidated to a single color-changing function with parameter.

Version 5: Event Listeners in JavaScript

<script> let square = document.getElementById('square'); square.addEventListener('click', (event) => changeColour('green')); square.addEventListener('mouseover', (event) => changeColour('gray')); square.addEventListener('mouseout', (event) => changeColour('red')); function changeColour(colour) { square.style.background = colour; } </script>

Moved event handlers from HTML to JavaScript using addEventListener.

Version 6: Dynamic Content Loading

<body onload="greeting()"> <div id="words"><!-- Text Will Go Here --></div> <script> function greeting() { words.innerHTML = "Welcome to Flatland.<br> I am Square."; } </script>

Added dynamic content loading on page load.

Version 7: JavaScript-Only Event Binding

<script> document.onload = greeting(); function greeting() { words.innerHTML = "Welcome to Flatland.<br> I am Square."; } </script>

Moved all event binding to JavaScript (no onload in HTML).

Version 8: External JavaScript File

<script src="script.js"></script>

Final version with all JavaScript in external file (simulated here).
